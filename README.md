# Sign-Language-Detection
Programmed an application that allows the client to make certain sign-language gestures in front of a webcam and then prints the corresponding letter.

## Technologies Used
- `Python 3.7`
- `TensorFlow 2.3.0`
- `Numpy 1.18.5`
- `OpenCV 4.4.0.42`
- `Keras 2.4.3`
- `Keras-Preprocessing 1.1.2`
